Private cloud

But instead of develop a dumb app we will create a web app where users can upload and download pics, like a standard cloud service.

Technologies :
    Java Spring
    JPA
    Vue.js / React
    
Infrastructure:
    GitHub CI/CD
    Docker
    Kubernetes
    Rancher

    user : name, id, mdp
    pics : name, date, album
    album: name, [pics]

Vue : 
    Album 
    All pics
    Pics by user
    Pics by mounth
    Upload 
    Download per pics
